# ตัวอย่างใช้กับกราฟ XAUUSD ที่ส่งมา

อ่านกราฟ:
- D1: Bias ลง
- H4: ราคาอยู่ใกล้แนวรับ
- สรุป: ยังไม่ Sell ไล่ตรงแนวรับ

แผนที่ควรใส่ใน Web App:
Step 1:
- Symbol: XAUUSD
- Direction: SELL
- TF: H4
- Order Type: Sell Stop หรือ Sell Limit ตามแผน

Step 2:
- ถ้ามีข่าวแดง/ส้ม USD: เลือก Red/Orange แล้วระบบจะล็อก “ยังไม่เข้า”

Step 3:
- D1 Trend: ลง
- H4 Position: ติดแนวรับ
- Bias: เน้นรอ Sell
- Support: 4430-4440, 4330
- Resistance: 4488-4495, 4520-4535
- แผน: D1 ลงแต่ H4 ติดแนวรับ จึงไม่ Sell ไล่ ต้องรอหลุด 4430 แล้ว Retest หรือเด้งขึ้นแนวต้านแล้วมี Bearish Trigger

Step 4:
- ตอนนี้อย่าเพิ่งติ๊ก Trigger ถ้ายังไม่มีแท่งยืนยัน
- อย่าเพิ่งติ๊ก “ไม่ Sell บนแนวรับ” ถ้ายังจะ Sell ตรงนี้ เพราะข้อนี้ไม่ผ่าน

Step 5:
- ใส่ Entry/SL/TP เฉพาะเมื่อมีแผนจริง
- RR ต้องอย่างน้อย 2R

Step 6:
- แนบรูป D1 และ H4
- บันทึกเหตุผลว่า “ยังไม่เข้า เพราะ H4 ติดแนวรับ”

Step 7:
- ถ้าระบบขึ้น “ยังไม่เข้า” ให้บันทึกเป็น Journal ได้ เพื่อเก็บเหตุผลว่าทำไมไม่เทรด
